## How to Use

### symbols
```bash
# -> Black Text(Enclosed black text with # both sides)
$ -> Left Side Space in New Line

``` 
### example

Paste you text into "Content.txt" for example.

```bash
$       #(Center Text in Black)#
this is my text in blue.
```
